# default_website
website with optional shopping mall
AETERNO 코스메틱 프로젝트 개발 최종 요약서
본 문서는 Vite + Vanilla JS + LocalStorage 기반으로 구축된 AETERNO 프리미엄 화장품 브랜드 기업 사이트 및 독립형 쇼핑몰 통합 개발 명세서입니다.

1. 프로젝트 아키텍처 개요
기술 스택: Vite, Pure Vanilla JS, Vanilla CSS, LocalStorage (Database Service)
포트 및 환경: 로컬 서버 기본 포트 3000 (vite.config.js)
디자인 테마: Rose Gold (hsl(25, 45%, 72%)) & Deep Onyx/Emerald 기반 글래스모피즘 프리미엄 디자인
데이터 보존: LocalStorageDbService를 사용하여 서버 연결 없이도 완벽한 비즈니스 로직(인증, 쇼핑, 주문, 문의, CMS)이 로컬 브라우저에서 영구 보존되도록 구성
2. 웹사이트 메뉴 및 SPA 라우팅 명세 (#/)
클라이언트 해시 기반 라우터(src/main.js)를 통해 화면 이동 시 깜빡임 없이 부드러운 스크롤 및 화면 전환을 제공합니다.

2.1 기업 브랜드 소개 사이트 (Default)
기본 모드에서는 회사 소개와 홍보 중심의 페이지로 노출됩니다.

메인 (#/): 메인 Hero 배너, 시그니처 3대 상품 쇼케이스, 브랜드 스토리 요약, 최신 보도자료 2선, 쇼핑몰 바로가기 안내 배너.
회사소개:
대표이사 인사말 (#/about/ceo): 대표이사의 신년사 및 인물 컷 프로필.
회사정보 (#/about/info): 회사명, 대표자명, 사업자번호, 연락처, 이메일 등의 기본 개요 정보.
인재채용 (#/about/careers): 바이오 연구소, 마케팅 본부 등 부서별 활성 채용 공고(모집중/마감 자동 라벨링).
제품소개:
제품 목록 (#/products): 가격이나 구매 버튼이 없는 순수 조회용 제품 안내 그리드.
제품 상세 (#/products/:id): 성분 및 핵심 기술 정보를 제공하는 조회 전용 상세 정보.
미디어랩:
보도자료 (#/media/press): 에테르노 언론 보도 목록 및 YouTube 브랜드 영상관.
갤러리 (#/media/gallery): 브랜드 룩북 비주얼 아카이브 및 브로셔 PDF 자료실 다운로드.
Contact Us:
문의게시판 (#/contact/inquiry): 이름, 이메일, 제목, 내용을 기재하는 1:1 고객 실시간 문의 등록 양식.
오시는 길 (#/contact/map): 본사 약도 및 구글 지도 iframe 반응형 임베드.
2.2 독립 쇼핑몰 영역 (Shop Mode)
관리자 시스템 설정에서 쇼핑몰 활성화를 ON할 경우 활성화되는 영역입니다.

쇼핑몰 홈 (#/shop): 상품 판매 목록, 할인율 라벨, 장바구니 담기 액션.
상품 상세 (#/shop/product/:id): 수량 조절 버튼, 바로구매 및 장바구니 담기.
회원 관리:
로그인 (#/shop/login): 쇼핑 회원 로그인.
회원가입 (#/shop/register): 배송지 정보, 연락처 등을 기재하는 자사몰 회원가입.
마이페이지 (#/shop/mypage): 로그인 회원 전용 주문/배송 현황 추적.
주문조회 (#/shop/orders): 비회원 주문 전용 주문번호 + 연락처 통합 매칭 조회.
결제 단계:
주문서 작성 (#/checkout): 배송 요청사항 기재 및 최종 결제 금액 산정(무료배송 임계치 도달 시 배송비 자동 무료 처리).
결제 완료 (#/order-complete): 주문 완료 메시지 및 가상 입금 계좌 안내.
3. 통합 어드민 대시보드 CMS (#/admin/dashboard)
최초 관리자 인증 정보:
아이디: siteadmin
초기 비밀번호: !admin1004 (Subtle Crypto SHA-256 비동기 암호화 기반 최초 로그인 시 필수 변경 강제 모달 연동)
7가지 관리 제어판 기능:
대시보드: 신규 대기 주문, 미답변 문의, 총 회원수, 누적 매출 통계 데이터 보드.
콘텐츠 관리: 인사말, 회사정보, 채용 공고(추가/삭제), 보도자료(추가/삭제), 갤러리 룩북(추가/삭제), Hero 및 푸터 상세 텍스트 실시간 동기화.
회원 관리: 자사 쇼핑몰에 가입한 전체 일반 회원 명부 조회.
상품 관리: 쇼핑몰 상품 카테고리, 판매가, 정가, 재고량, 품절(Sold Out) 수동 활성화 설정 및 상품 추가/삭제.
주문 관리: 고객 주문 정보, 배송지 확인 및 결제/발송 단계(대기→입금확인→배송중→배송완료→취소) 관리.
문의 관리: 1:1 고객 Inquiry 상세 내용 룩업 및 답글 저장(상태값 대기/완료 실시간 매핑).
시스템 설정: 쇼핑몰 기능 활성화 스위치 (ON/OFF), 기본 배송비, 무료배송 임계치, 무통장 결제 계좌 관리.
4. 데이터베이스 및 저장 구조
LocalStorage 내 다음의 독립된 키 세트를 구성하여 완벽한 데이터 격리 및 정합성을 구현했습니다.

site_contents: 메인 및 서브 페이지 콘텐츠 전체 (CEO, 정보, 채용, 상품, 미디어, 보도, 갤러리)
admin_settings: 어드민 계정 ID, 비밀번호 해시, 변경 여부 플래그
shop_settings: 쇼핑몰 활성화 여부, 배송비 및 계좌 정보
shop_users: 가입 쇼핑몰 유저 객체 배열
site_inquiries: 1:1 Inquiry 접수 및 답변 객체 배열
shop_orders: 쇼핑몰 주문 상세 내역 및 배송 상태 정보 객체 배열
shop_cart: 현재 쇼핑몰 사용자의 장바구니 보관 아이템 객체 배열
5. 핵심 UX 및 기술 트러블슈팅
드롭다운 이탈 방지 브릿지: 회사소개, 미디어랩, Contact Us와 같이 높이가 있는 드롭다운 메뉴 이동 시 호버 상태를 잃지 않도록 메뉴 상단에 20px 크기의 투명한 가상 브릿지(.dropdown-content::before) 영역을 구축하여 끊김 현상을 근본적으로 해결했습니다.
쇼핑몰 On/Off 동적 가시성: 쇼핑몰 상태 토글에 따라 네비게이션 헤더 및 회원 권한, 장바구니 아이콘, 메인 홈 배너 등이 즉시 반영되는 동적 렌더링 로직을 완벽히 이행했습니다.
Vite 프로덕션 빌드 최적화: 13개 핵심 모듈이 npx vite build 시 에러 없이 성공적으로 정적 assets로 컴파일 완료되었습니다.
